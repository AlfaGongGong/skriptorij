BOOKLYFI TURBO CHARGED
AI-Powered Book Translation & Refinement Engine
================================================================================

1. SAZETAK APLIKACIJE
--------------------------------------------------------------------------------
BOOKLYFI je napredni sistem za automatsko prevodjenje i lekturu EPUB/MOBI knjiga
sa engleskog na bosanski/hrvatski jezik. Koristi flotu od 50+ besplatnih API
kljuceva rasporedjenih preko vise AI provajdera (Gemini, Groq, Mistral,
Sambanova, OpenRouter, Cohere, Chutes, HuggingFace, Kluster) sa automatskim
failover-om i pametnim rutiranjem.

Kljucne funkcionalnosti:
- Upload i odabir knjiga (EPUB/MOBI)
- Vise AI provajdera sa automatskim prebacivanjem
- Live dashboard sa progresom, ETA, pipeline koracima
- Fleet menadzer sa health barovima, toggle dugmadima
- Checkpoint sistem - nastavlja obradu od mesta prekida
- Neon animacije na BOOKLYFI naslovu
- Potpuno responzivan - radi na desktopu i Androidu

2. NACIN RADA
--------------------------------------------------------------------------------
2.1 Frontend (HTML/CSS/JS)
- index.html - Glavni interfejs sa wizardom (2 koraka), dashboardom i tabovima
- style.css - Dark tema sa glassmorphism efektima, neon akcentima, responzivan
- app-clean.js - Sva logika: upload, odabir knjiga, pokretanje obrade, dashboard
  polling, fleet panel, ETA, pipeline animacija

2.2 Backend (Flask + Python)
- main.py - Ulazna tacka, pokrece Flask server
- app.py - Flask aplikacijska fabrika, definise rute
- api_fleet.py - Fleet Manager: upravljanje API kljucevima, health scoring,
  automatsko deaktiviranje
- skriptorij.py - Glavni engine za obradu knjiga (monolit, 3600+ linija)
- run.py - Wrapper za modularni engine (importuje module)

2.3 Modularni engine (core/, network/, processing/, epub/, utils/)
- core/ - Osnovne klase i funkcije (engine, text_utils, quality, prompts,
  chunking)
- network/ - HTTP klijent, provider router, URL generatori, rate limiter
- processing/ - Pipeline za obradu (pipeline, workers, parallel, rescue, retro)
- epub/ - EPUB parser, stilizovanje, pakovanje
- utils/ - Logovanje, ciscenje checkpointa, file utilities

2.4 Tok obrade
1. Korisnik uploaduje ili odabira EPUB/MOBI knjigu
2. Odabira model (V8_TURBO koristi sve provajdere)
3. Klikom na "Pokreni Sistem" backend:
   - Raspakuje EPUB u radni direktorijum
   - Analizira knjigu (zanr, ton, likovi, glosar)
   - Deli HTML fajlove na chunkove od ~800 reci
   - Svaki chunk prolazi kroz pipeline:
     - Prevodilac (brzi modeli: Groq, Cerebras)
     - Validator (provera semanticke vernosti)
     - Lektor (kvalitetni modeli: Gemini, Mistral)
     - Korektor (gramatika)
     - Guardian (konzistentnost)
     - Polish (finalni stil)
   - Rezultati se kesiraju u .chk fajlove
   - Na kraju se pakuje novi EPUB
4. Frontend prikazuje progres u realnom vremenu

2.5 Fleet Manager
- Svaki API kljuc ima health score (RPM + RPD)
- Automatsko deaktiviranje nakon 3 greske u 30s
- 402/403/401 -> trajno deaktiviranje
- 429 sa retry-after > 1h -> dnevni limit, deaktiviranje na 23h
- Toggle dugmad za rucno ukljucivanje/iskljucivanje

5. KAKO POKRENUTI
--------------------------------------------------------------------------------
1. Instalirati zavisnosti:
   pip install -r requirements.txt

2. Konfigurisati API kljuceve u dev_api.json

3. Pokrenuti server:
   python main.py

4. Otvoriti browser:
   http://127.0.0.1:8080
